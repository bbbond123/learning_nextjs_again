This is an example MDX page that demonstrates the power of combining **Markdown** with **React components**.

## What is MDX?

MDX allows you to write **JSX** directly in your Markdown documents. This means you can:

- Write content in Markdown
- Use React components
- Create interactive documentation
- Build dynamic content

## Features

### 1. Markdown Syntax

You can use all standard Markdown features:

- **Bold text**
- *Italic text*
- `Inline code`
- [Links](https://nextjs.org)

### 2. Code Blocks

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('MDX'));
```

### 3. Custom Components

You can also use React components directly in MDX:

<div className="p-4 bg-blue-100 border-l-4 border-blue-500 rounded my-4">
  <p className="font-bold">Info Box</p>
  <p>This is a custom styled component rendered in MDX!</p>
</div>

## Getting Started

To create your own MDX pages:

1. Create a new `.mdx` file in your `app` directory
2. Write your content using Markdown and JSX
3. Navigate to the page in your browser
