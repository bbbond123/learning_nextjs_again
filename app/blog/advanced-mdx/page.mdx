# MDX 高级特性演示 🚀

MDX 不仅仅是 Markdown + JSX，它是创建交互式文档的强大工具。

---

## 1. 交互式组件 🎮

在 MDX 中可以嵌入完全交互式的 React 组件：

<InteractiveCounter />

**代码示例：**
```jsx
<InteractiveCounter />
```

这个计数器是一个完整的 React 组件，支持状态管理和事件处理！

## 5. 变量和表达式 🔢

MDX 支持 JavaScript 表达式：

export const author = 'MDX 大师'
export const year = 2024
export const technologies = ['React', 'Next.js', 'TypeScript', 'MDX']

- **作者**: {author}
- **年份**: {year}
- **当前时间**: {new Date().toLocaleString('zh-CN')}
- **技术栈数量**: {technologies.length}
- **随机数**: {Math.floor(Math.random() * 100)}

**定义变量：**
```jsx
export const author = 'MDX 大师'
```

**使用变量：**
```jsx
作者是：{author}
```

## MDX 的核心优势 ✨

1. **Markdown 的简洁性** - 快速编写内容
2. **React 的强大性** - 完整的组件生态系统
3. **TypeScript 支持** - 类型安全
4. **完全可定制** - 自定义组件库
5. **服务端渲染** - Next.js 完美集成
6. **开发体验** - 热重载、代码提示
